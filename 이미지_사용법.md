# 이미지 추가 방법 📸

## 1. 이미지 파일 저장 위치
이미지 파일은 `public/images/` 폴더에 저장하세요.

예: `/public/images/meme1.jpg`

## 2. questions.js에서 이미지 추가하기

문제 데이터에 `image` 필드를 추가하면 됩니다:

```javascript
{
  id: 1,
  type: 'multiple-choice',
  question: '다음 중 사진의 밈은?',
  image: '/images/meme1.jpg',  // ← 이렇게 추가!
  options: [
    { id: 'a', text: '선택지1', isCorrect: false },
    { id: 'b', text: '선택지2', isCorrect: true },
    // ...
  ],
  explanation: '설명...'
}
```

## 3. 이미지가 없는 문제는?
이미지가 없는 문제는 `image: null` 또는 `image` 필드를 생략하면 됩니다.

## 4. 예시

```javascript
{
  id: 5,
  type: 'multiple-choice',
  question: '다음 중 사진의 밈이 맞는 것은?',
  image: '/images/차단각.jpg',  // public/images/차단각.jpg 파일이 있다고 가정
  options: [
    { id: 'a', text: '차단각', isCorrect: true },
    { id: 'b', text: '블록각', isCorrect: false },
    { id: 'c', text: '신고각', isCorrect: false },
    { id: 'd', text: '차단당각', isCorrect: false }
  ],
  explanation: '차단각이 맞아요!'
}
```

## 5. 권장 이미지 형식
- 파일 형식: JPG, PNG, GIF, WebP
- 권장 크기: 너비 600px 이하 (모바일 최적화)
- 용량: 200KB 이하 권장 (빠른 로딩)

## 6. 이미지 경로 규칙
- `public` 폴더의 파일은 `/`로 시작하는 경로로 접근합니다
- 예: `public/images/test.jpg` → `/images/test.jpg`
- 절대 경로나 상대 경로(`../`)는 사용하지 마세요
